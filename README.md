## 👨‍💻 Meu Terminal

Uma aplicação que simula um terminal, com diversas funcionalidades informativas e divertidas.

## 📝 Comandos

|             Comando              |      Output       |
| :------------------------------: | :---------------: |
|               help               | Exibe os comandos |
| clear \|\| cls \|\| Ctrl+Shift+L |   Limpa a tela    |
